# 기능 명세: 스마트 사이클 (Smart Cycle)

## 1. 기능 개요
창의 크기를 단축키 반복 입력을 통해 자연스럽게 순환시키는 기능.
추가 단축키 설정 없이 직관적으로 세밀한 창 크기 조절이 가능하다.

## 2. 동작 방식
특정 방향 단축키를 반복 입력할 경우 창의 너비가 다음 순서로 순환된다.

**순환 비율**: `1/2 → 1/3 → 2/3 → 1/2 → ...`

### 예시 (좌측 절반 단축키 입력 시)
| 입력 횟수 | 결과 |
|---|---|
| 1회 | 좌측 1/2 |
| 2회 | 좌측 1/3 |
| 3회 | 좌측 2/3 |
| 4회 | 좌측 1/2 (반복) |

## 3. 구현 상세
- 현재 창의 크기와 위치를 조회하여 `is_nearly_equal` 함수로 현재 상태가 어느 단계인지 판별 후 다음 단계로 전환
- 판별 오차 허용 범위: ±20px (모니터 해상도에 무관하게 동일 적용)
- `window_controller.go`에서 순수 Go 로직으로 구현 (CGo 불필요)

## 4. 엣지 케이스
- **커스텀 비율 모드**: 커스텀 비율(예: 60%)은 사이클에 포함하지 않고 단순 적용
- **최소 창 크기 제한**: 앱의 최소 폭이 목표 비율보다 클 경우, 창 이동 후 실제 적용된 크기로 좌표 재보정(Re-anchoring) 필요
  - 우측 정렬 시: `x = 모니터우측경계 - 실제창너비` 로 재계산
