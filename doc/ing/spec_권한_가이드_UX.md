# 기능 명세: macOS 권한 부여 2단계 가이드

## 1. 개요
단축키 기능이 정상 작동하려면 macOS에서 두 가지 권한이 모두 필요하다.
웹 UI 가이드 화면에서 두 권한을 모두 획득하도록 단계별로 안내한다.

## 2. 필수 권한
| 단계 | 권한 | 설정 경로 |
|---|---|---|
| 1단계 | 손쉬운 사용 (Accessibility) | 시스템 설정 > 개인 정보 보호 및 보안 > 손쉬운 사용 |
| 2단계 | 입력 모니터링 (Input Monitoring) | 시스템 설정 > 개인 정보 보호 및 보안 > 입력 모니터링 |

> **주의**: 권한 부여 후 반드시 앱을 종료하고 다시 실행해야 반영됨.

## 3. 웹 UI 가이드 흐름
1. 앱 실행 시 권한 상태 자동 확인 (`/api/status` 응답에 `accessibility_granted`, `input_monitoring_granted` 포함)
2. 미승인 권한이 있으면 가이드 팝업 노출
3. 각 단계 완료 시 체크 표시로 진행 상황 시각화
4. 모든 권한 완료 후 "앱을 다시 실행해 주세요" 강조 안내

### 가이드 단계
```
[1단계] 손쉬운 사용 설정  →  [완료 ✓]
[2단계] 입력 모니터링 설정  →  [완료 ✓]
[3단계] 앱 재실행 (필수)
[완료]
```

## 4. 백엔드 구현 포인트
- `checkAccessibilityPermission()` — CGo + `AXIsProcessTrusted()`
- `checkInputMonitoringPermission()` — CGo + `CGPreflightListenEventAccess()`
- `/api/status` 응답에 두 권한 상태 필드 포함

## 5. 알려진 이슈
- 일부 macOS 버전에서 '손쉬운 사용' 권한만으로도 작동하는 것처럼 보이지만, 보안 패치 후 갑자기 동작하지 않을 수 있음 → 두 권한 모두 필수로 안내할 것
