# 요구사항 명세서

## 1. 프로젝트 개요
macOS 환경에서 전역 단축키를 이용해 활성 윈도우의 크기와 위치를 자유롭게 조절하는 생산성 유틸리티.
Magnet 앱과 유사한 사용자 경험을 제공하며, 브라우저 기반 웹 UI를 통해 단축키를 사용자가 직접 설정할 수 있다.

## 2. 핵심 기능 요구사항

### 2.1 윈도우 제어 모드
- **1/2 분할:** 좌측, 우측, 위쪽, 아래쪽 절반 배치
- **1/3 분할:** 좌측, 중앙, 우측 1/3 배치
- **2/3 분할:** 좌측, 우측 2/3 배치
- **1/4 분할 (Corner Snapping):** 좌상, 우상, 좌하, 우하 구석 배치
- **커스텀 비율:** 사용자가 설정한 비율(%)로 좌/우/상/하 배치
- **스마트 사이클 (Smart Cycle):** 동일 단축키 반복 입력 시 1/2 → 1/3 → 2/3 → 1/2 순환
- **특수 모드:** 최대화, 이전 상태로 복구 (Restore)
- **멀티 모니터:** 활성 창이 위치한 모니터의 작업 영역(MenuBar 제외) 기준으로 좌표 산출

### 2.2 사용자 인터페이스
- **메뉴바 트레이:** 메뉴바 아이콘 클릭으로 설정 UI 접근
- **웹 UI 설정창:** 브라우저 기반 단축키 설정 페이지
- **단축키 설정:** 버튼 클릭 후 키 조합 입력으로 즉시 변경
- **단축키 삭제:** 단축키 녹화 중 `Backspace` 또는 `Delete` 키로 초기화
- **실시간 적용:** 단축키 변경 시 리스너 재시작으로 앱 재실행 없이 즉시 반영
- **접근성 권한 가이드:** 앱 실행 시 권한 상태 자동 확인 및 2단계 안내 (손쉬운 사용 + 입력 모니터링)

### 2.3 데이터 관리
- **설정 저장:** `config.json`으로 로컬 영구 저장 및 로드
- **기본값 복구:** API를 통한 `default-config.json` 기반 초기화

### 2.4 배포
- **단일 바이너리:** `go build` 한 줄로 빌드, 별도 런타임 불필요
- **앱 번들:** `.app` 번들 + `.dmg` 설치 이미지 제공
- **Universal Binary:** Intel + Apple Silicon 통합 바이너리 (`lipo`)

## 3. 기술 스택
- **언어:** Go 1.22+ (CGo 포함)
- **단축키 감지:** CGo + Carbon `RegisterEventHotKey`
- **창 제어:** CGo + ApplicationServices (AX API)
- **멀티모니터:** CGo + Cocoa `NSScreen`
- **트레이 UI:** `github.com/getlantern/systray`
- **웹 서버:** `github.com/gin-gonic/gin`
- **설정 파일:** `encoding/json` 표준 라이브러리

## 4. 제약 사항
- macOS 12.0 이상 전용 (Apple Silicon 및 Intel Mac 지원)
- 시스템 접근성(Accessibility) 권한 필수
- 입력 모니터링(Input Monitoring) 권한 필수
- CGo 빌드를 위한 Xcode Command Line Tools 필요
