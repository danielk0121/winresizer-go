
1. 파이썬을 활용한 macOS 윈도우 제어 (기술 스택 유사)
   직접적인 파이썬 윈도우 매니저 구현체를 살펴보고 싶을 때 유용한 프로젝트들입니다.
   - macpywm (h-ohsaki/macpywm 등): 파이썬으로 작성된 macOS 윈도우 매니저입니다. 
     다만 파이썬만으로 윈도우를 직접 제어하기보다는, C로 짜여진 강력한 매니저인 Yabai를 시스템 단에 깔아두고 파이썬이 
     이를 래핑(Wrapping)하여 단축키와 레이아웃을 제어하는 방식을 취하고 있습니다.
   
   - PyGetWindow (asweigart/PyGetWindow): 유명한 크로스 플랫폼 파이썬 윈도우 제어 라이브러리입니다. 
     macOS 전용은 아니지만, 소스 코드 내부를 보면 macOS 환경에서 윈도우 크기와 위치를 어떻게 찾아내고 조작하는지 파악하기 매우 좋습니다.

2. 마그넷(Magnet) 기능의 완벽한 오픈소스 구현체 (동작 유사)
   파이썬이 아닌 Swift/Objective-C 기반이지만, macOS Accessibility API를 사용해 '활성 윈도우 핸들링 및 좌표 계산'을 
   어떻게 하는지 네이티브 코드를 참고하고 싶을 때 봐야 할 교과서 같은 프로젝트들입니다.
   - Rectangle (rxhanson/Rectangle): 현재 macOS 생태계에서 마그넷을 완벽하게 대체하고 있는 가장 대중적이고 압도적인 Swift 기반 오픈소스입니다.
   
   - ShiftIt (fikovnik/ShiftIt): 단축키를 이용한 창 1/2, 1/4 분할 유틸리티의 원조격인 프로젝트입니다. 
     구조가 직관적이어서 기본 윈도우 API 흐름을 파악하기 좋습니다.
